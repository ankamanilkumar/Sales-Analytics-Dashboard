# 📊 Sales Analytics Dashboard (Python)

## 📌 Project Overview

This project analyzes sales data using Python. It demonstrates data cleaning, exploratory data analysis (EDA), KPI calculation, and data visualization to identify sales trends and business insights.

---

## 🎯 Objectives

- Clean raw sales data
- Perform exploratory data analysis
- Calculate business KPIs
- Create sales visualizations
- Generate business insights

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Microsoft Excel
- Visual Studio Code

## 📊 Dataset

The dataset contains:

- Order ID
- Order Date
- Customer Name
- Category
- Sales
- Profit

---

## 🧹 Data Cleaning

The following operations were performed:

- Removed duplicate records
- Checked missing values
- Converted date columns
- Created Month, Year, Quarter, and Day columns
- Exported cleaned dataset

---

## 📈 Exploratory Data Analysis

The project calculates:

- Total Sales
- Total Profit
- Total Orders
- Average Sales
- Profit Margin

Analysis includes:

- Category-wise Sales
- Category-wise Profit
- Monthly Sales
- Top Customers

## 💡 Business Insights

- Identified top-selling categories.

- Identified high-value customers.
- Compared sales and profit across categories.

## ▶️ How to Run

1. Clone the repository.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run data cleaning:

```bash
cd Python
python data_cleaning.py
```

4. Run EDA:

```bash
python eda.py
```
